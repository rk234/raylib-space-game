Basic endless scrolling game built with raylib to get familiar with C.

## Building

Uses CMake build system and clang compiler.

```
$ cd raylib-space-game
$ mkdir build && cd build
$ cmake ..
$ cmake --build build
```

## Running

```
$ cd raylib-space-game
$ build/raylib-space-game
```
